 ## Description

    
 This app uses npm express to for front end and backend routing which allows a client to take notes and save them by persisting the note via a json file. 

    
 ## Installation

    
 npm i

    
 ## Usage

    
 This app provides a convenient way to take notes to help with a busy schedule.
